# 🌟 Personal Life Coaches Website
A modern, responsive Life Coach Directory website built using **HTML5, CSS3, and JavaScript**. This project allows users to browse life coaches, search and filter coaches, view detailed coach profiles, contact the team, and enjoy a fully responsive experience across desktop, tablet, and mobile devices.

# ⚙️ Installation

Clone the repository
```bash
git clone https://github.com/nuthigauri21/life-coach-website.git
```

Go to the project folder
```bash
cd life-coach-website
```

Open
```text
index.html
```

in your browser.

# 🌐 Deployment

This project is deployed using **GitHub Pages**.
To deploy your own version:
1. Fork or clone this repository.
2. Push your changes to GitHub.
3. Go to **Settings** → **Pages**.
4. Select the **main** branch.
5. Save your settings.

Your site will be published automatically.
